# 勤怠管理アプリ

## 概要
Ruby on Railsで作成中の勤怠管理アプリ。

## 目的
- Rails基礎理解
- CRUD理解
- 勤怠ロジック実装
- Git/GitHub運用練習

## 主な機能
- 出勤打刻
- 退勤打刻
- 勤怠一覧表示
- 勤務時間計算
- 設定テーブルによる勤怠制御

## 使用技術
- Ruby
- Ruby on Rails
- SQLite3
- GitHub
